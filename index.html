    match /artifacts/{appId}/public/data/{document=**} {
      allow read, write: if true;
    }
  }

Laporan Detektor Error

Kenapa websitemu berstatus "Luring (Offline)"?

Status offline terjadi karena koneksi ke server database Firebase gagal dilakukan. Silakan periksa daftar baris error di bawah untuk mengetahui akar masalahnya:

Daftar Diagnostik Server (Error Log):

Solusi Umum Penyebab Offline:

  1. Aktifkan Anonymous Auth: Di Firebase Console, buka menu Authentication -> Sign-in method -> Aktifkan "Anonymous". Jika ini tidak aktif, kamu tidak akan bisa login.
  2. Buat Firestore Database: Pastikan kamu sudah menekan tombol "Create Database" di menu Firestore Firebase Console.
  3. Atur Security Rules: Pastikan rules database-mu mengizinkan akses tulis dan baca untuk jalur `/artifacts/...`
